IVAutofill Toolkit
One-Click Application Filling

Career Dispatch — Chrome Extension

The browser extension runs on every job application page. Press ⌘⇧F (Mac) or Ctrl+Shift+F (Win/Linux) to autofill, or click the extension icon for manual field mapping.

Installation:
  1. Download the extension ZIP from this conversation
  2. Unzip to a folder you'll keep (the extension loads from this folder)
  3. Open chrome://extensions in Chrome/Edge/Brave
  4. Toggle Developer mode (top right)
  5. Click Load unpacked → select the unzipped folder
  6. Click the extension's icon → Import Profile → select the profile.json you exported above
  7. Done. Visit any application page and press ⌘⇧F / Ctrl+Shift+F

⚠ Re-export and re-import your profile whenever you update your dossier — the extension stores a local copy.

Extension Features
What it handles automatically
  • Smart field detection — matches labels, placeholders, name, id, aria-label, and ancestor label text via regex patterns
  • React/Vue compatibility — bypasses synthetic event issues using native setter injection
  • Select/dropdown matching — intelligently picks the closest option (e.g., "H-1B" matches "H1B Visa (requires sponsorship)")
  • EEO & demographic fields — handles voluntary self-ID fields on Greenhouse/Lever/Workday
  • Visual confirmation — filled fields flash yellow, a summary toast shows field count
  • Skips already-filled fields — non-destructive; won't overwrite your edits
  • Manual picker mode — click extension icon → pick a specific field to fill individually
